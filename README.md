

### It's a questionnaire platform.


#### Features
- Post questions
- Voting system
- Collections
- Global data search
- AI generated answers
- Light/dark mode 


#### Technologies Used
-  ***Next.js 14***
-  ***TypeScript*** 
-  ***TailwindCSS*** 
-  ***MongoDB*** 
- ***Clerk Auth***
- ***GeminiAi*** 
- ***Zod*** 
- ***ESLint*** 
- ***Prettier*** 
- ***Shadcn***
